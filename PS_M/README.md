## 📝 وصف المشروع

**FRINDES** هو تطبيق مكتبي ذكي ومفتوح المصدر لإدارة محل بلايستيشن صغير إلى متوسط.  
تم تطويره باستخدام: **React + TypeScript + Supabase**

يهدف إلى:
- تبسيط إدارة الجلسات والمبيعات.
- تقليل الخطأ البشري.
- توفير لوحة تحكم تفاعلية لصاحب المحل.
